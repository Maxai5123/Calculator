
# Проект "Калькулятор"

## Описание проекта
Просто калькулятор

## Цели проекта
- помощь в решении простых и сложных примеров, в несколько действий

## Используемые технологии
- `HTML` для разметки и подключения функций калькулятора.
- `CSS` для создания стилей калькулятора.
- `JavaScript` для описания функций калькулятора.

## Функции проекта
Для работы кнопок используется событие `onclick`, которое запускает функции:
- `appendNumber()` - функция, которая добавляет на экран калькулятора цифры, которые вводит пользователь (цифры от `0` до `9`). Она принимает на вход аргумент `number`.
- `appendOperator()` - функция, которая добавляет на экран калькулятора оператор (действие) которое вводит пользователь (`+`,`-`,`*`,`/`). Она принимает на вход аргумент `operator`.
- `clearDisplay()` - функция, которая очищает экран калькулятора. Для использования нажмите `C`. На вход она не принимает аргументов.
- `calculateResult()` - функция, которая вычисляет выражение, выведенное на экране калькулятора. Для использования нажмите `=`. На вход она не принимает аргументов.

## Пример запуска
Для запуска проекта установите все файлы на свой компьютер, и откройте файл `index.html`

## Пример использования
- Если вы хотите ввести какое либо число или цифру, просто найдите и нажмите на нужные вам цифры от `0` до `9`.
- если вы хотите выполнить какое-либо действие, выберите оператор (арифметическое действие) `+`,`-`,`*`,`/`.
- для получения результата выражения, написанного вами на экране, нажмите `=`.
- если вы хотите очистить экрун от вычислений, нажмите на `C`.
